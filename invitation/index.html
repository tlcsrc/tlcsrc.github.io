Current Initiatives

a HoTT Programming Language

We would like a practical programming language based on Univalent Foundations—the new foundation of mathematics. Of course, firstly we must understand what that is. To this end we are currently walking through Robert Harper’s lectures on Higher Type Theory (aka HoTT). Hopefully it will be possible to integrate ideas of Higher Type Theory into existing programming languages such as Idris and Haskell. Failing that, the Triplet programming language will be born.

Haskell ⇔ Swift

The equivalent of language-c-line for Swift.

  • Write a Swift parser in Haskell, language-swift-quote.
  • Build Swift quasiquotation, language-swift-inline.

Systems Programming with Haskell

  • The Go programming language is becoming increasingly popular in the systems/infrastructure/cloud space. We’d like to show that the Glorious GHC is already highly suited to this application. To demonstrate this, we’ll do some ports of popular Go project in Haskell. e.g. etc.

  • Using Turtle and Nix to port Bash scripts to Haskell scripts.

Nix in Haskell

Nix is a purely functional package system but it’s currently written in a combination of C and Perl. Some kind-hearted folks have already started to remedy this situation and we’d like to help. See the hnix project on GitHub.

Truffled PureScript

It would be nice to have a principled programming language that was practical to use on the JVM. PureScript is such a language and Truffled PureScript is an endeavour to bring about an implementation of PureScript on the JVM using Graal/Truffle1. We’d like to help.
